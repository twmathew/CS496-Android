# CS496-Android
Contains 3 Assignments:

SQLite/Location App:
Implements permission handling, location access and database creation/access.
Demo Video Link: https://media.oregonstate.edu/media/t/1_4p8uispj

Google + App:
Using OAuth 2.0, connects to the Google+ Domains API, allows posting of simple text posts via and displays the 3 most recent posts made.
Demo Video Link: https://media.oregonstate.edu/media/t/1_ihf1l6xw

Final Project:
Connects to a 3rd party API, allows the user to log in, view data via the API, and create data for storage on a SQLite Database.
Demo Video Link: https://media.oregonstate.edu/media/t/1_yu8ys5nh
